# Pola Raya Studio Frontend Starter Template

Modern frontend starter template built with:

- React 19
- Vite 8
- TypeScript 6
- Tailwind CSS v4
- shadcn/ui
- TanStack Router
- Zustand
- ESLint (flat config)
- Prettier + Tailwind class sorting

Designed as a clean, scalable baseline for frontend-heavy applications.

---

# Stack

## Core

- React 19
- Vite 8
- TypeScript 6

## Styling & UI

- Tailwind CSS v4
- shadcn/ui
- Radix UI
- Lucide React
- Sonner
- Geist Variable Font
- tw-animate-css

## Routing

- TanStack Router
- File-based routing
- Auto-generated route tree

## State Management

- Zustand

## Developer Experience

- ESLint (flat config)
- Prettier
- prettier-plugin-tailwindcss
- TypeScript strict mode
- Path aliases (`@/*`)
